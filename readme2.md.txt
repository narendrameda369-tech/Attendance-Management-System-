## Project Overview

The Attendance Management System is a Python-based desktop application developed using the Tkinter GUI library. The application helps manage student attendance records efficiently through a simple and user-friendly graphical interface.

This system allows users to add student attendance details, mark attendance status (Present/Absent), search student records, delete attendance entries, and save attendance data for future use. The application also provides a real-time attendance summary dashboard.

---

## Objectives

* To automate the process of recording student attendance.
* To reduce manual errors in attendance management.
* To provide an easy-to-use graphical interface.
* To demonstrate Python GUI development and file handling concepts.

---

## Features

✅ Add Student Attendance Records

✅ Mark Students as Present or Absent

✅ Automatic Date and Time Recording

✅ Search Student by Roll Number or Name

✅ Delete Attendance Records

✅ Save Attendance Records to File

✅ Attendance Summary Dashboard

✅ User-Friendly Interface using Tkinter

## Working Procedure

1. Enter Student Roll Number.
2. Enter Student Name.
3. Select Attendance Status.
4. Click **Add Attendance**.
5. Attendance is displayed in the table.
6. Use Search to find a student.
7. Use Delete to remove records.
8. Click Save Attendance to store records permanently.
