# Attendance Management System

## Project Description

The Attendance Management System is a Python-based desktop application developed using the Tkinter GUI library. The system is designed to simplify the process of recording, managing, and maintaining student attendance records in an organized manner.

This application allows users to add student attendance details, mark students as Present or Absent, search attendance records, delete unwanted entries, and save attendance data for future reference. The system also records the date and time of each attendance entry automatically and provides a dashboard displaying the total number of students, present students, and absent students.

The project demonstrates the practical implementation of Python programming concepts such as GUI development, file handling, event-driven programming, data management, and user interaction. It provides an efficient and user-friendly solution for educational institutions to maintain attendance records digitally.

### Key Features

* Add student attendance records
* Mark students as Present or Absent
* Automatic date and time recording
* Search student records
* Delete attendance records
* Save attendance data to a file
* Attendance summary dashboard
* User-friendly graphical interface

### Technologies Used

* Python 3.x
* Tkinter (GUI Library)
* File Handling
* Datetime Module

### Project Outcome

This project helps automate attendance management, reduces manual paperwork, improves record accuracy, and provides a simple digital solution for tracking student attendance.
